Multi-Modal Encoder Architecture

Single-modality encoders with VSA/HDC projection
Figure 2. Detailed encoder architectures for each modality. Visual encoder uses ResNet-18 backbone with Feature Pyramid Network (FPN) outputting multi-scale features (2048, 1024, 512, 256 channels). Event encoder processes stream data through voxel grid representation with FPGA-compatible lightweight CNN. Audio encoder converts waveforms to mel-spectrograms via STFT. IMU encoder uses 1D CNN + bidirectional LSTM.
